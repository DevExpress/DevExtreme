# DevExtreme React Playground

## Run Examples

1. Install npm packages:

    ```bash
    npm install
    ```

2. Generate components and build scripts:

    ```bash
    npm run build
    npm run build:react
    ```

    or

    ```bash
    npm run build:react:watch
    ```

## Compile and Hot Reload an Application

```bash
cd ./playground/react
npm install
npm start
```
