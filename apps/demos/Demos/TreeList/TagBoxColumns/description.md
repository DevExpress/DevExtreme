In v25.2, we will add a built-in [TagBox](https://js.devexpress.com/Documentation/Guide/UI_Components/TagBox/Getting_Started_with_TagBox/) column to the DevExtreme TreeList component. These columns allow users to select multiple values when editing cell content or specify multiple search criteria within the Filter Row.
