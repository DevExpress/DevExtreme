DevExtreme [TreeView](/Documentation/Guide/UI_Components/TreeView/Overview/) ships with configurable selection capabilities. Users can click nodes or item checkboxes to select. This demo allows you to modify selection behavior using controls below the TreeView. A DevExtreme [List](/Documentation/Guide/UI_Components/List/Overview/) to the right of the TreeView displays selected items in a flat (non-hierarchical) list.
<!--split-->

This demo allows you to specify a [selection mode](/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/#selectionMode) for enabled and [disabled nodes](/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/#disabledNodeSelectionMode). You can configure the visibility of [selection checkboxes](/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/#showCheckBoxesMode), including the Select All checkbox. You can also toggle [selection on click](/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/#selectByClick) and [recursive selection](/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/#selectNodesRecursive).

To synchronize the List with TreeView, this demo implements a [selection change](/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/#onSelectionChanged) handler. The handler calls [getSelectedNodes()](/Documentation/ApiReference/UI_Components/dxTreeView/Methods/#getSelectedNodes) to update the List data source.

TreeView allows you to update node selection states programmatically. You can select/unselect items using [node keys](/Documentation/ApiReference/UI_Components/dxTreeView/Methods/#selectItemkey), [data objects](/Documentation/ApiReference/UI_Components/dxTreeView/Methods/#selectItemitemData), or [DOM elements](/Documentation/ApiReference/UI_Components/dxTreeView/Methods/#selectItemitemElement). To initialize nodes in the selected state, configure [selection fields](/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/#selectedExpr) in item objects (default: **selected**).

You can use selection and other data fields to customize items as needed. This demo implements [itemRender](/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/#itemRender) to combine text from multiple fields and does not specify a [text](/Documentation/ApiReference/UI_Components/dxTreeView/Configuration/items/#text) field in the TreeView data source.
