You can use tooltip API members to manage tooltip visibility in code. This demo creates a [DevExtreme PieChart](/Documentation/Guide/UI_Components/PieChart/Getting_Started_with_PieChart/) and disables its default tooltip. When you click a pie segment or select an item in the "Choose Region" box, a [showTooltip()](/Documentation/ApiReference/UI_Components/dxChart/Chart_Elements/Point/Methods/#showTooltip) method call displays the corresponding tooltip.
