DevExtreme Chart supports point crosshairs. When enabled, the component displays the value and argument of hovered points. To configure Chart crosshairs, specify the [crosshair](/Documentation/ApiReference/UI_Components/dxChart/Configuration/crosshair/) object.
<!--split-->

This demo configures the following **crosshair** properties:

- [color](/Documentation/ApiReference/UI_Components/dxChart/Configuration/crosshair/#color)    
Specifies crosshair line color.
- [width](/Documentation/ApiReference/UI_Components/dxChart/Configuration/crosshair/#width)    
Configures crosshair line width.
- [dashStyle](/Documentation/ApiReference/UI_Components/dxChart/Configuration/crosshair/#dashStyle)    
Defines the crosshair dash style.
- [label](/Documentation/ApiReference/UI_Components/dxChart/Configuration/crosshair/label/)    
Configures crosshair labels.

You can customize horizontal and vertical crosshair lines individually. To do this, configure the **crosshair**.[horizontalLine](/Documentation/ApiReference/UI_Components/dxChart/Configuration/crosshair/horizontalLine/) and **crosshair**.[verticalLine](/Documentation/ApiReference/UI_Components/dxChart/Configuration/crosshair/verticalLine/) objects.
