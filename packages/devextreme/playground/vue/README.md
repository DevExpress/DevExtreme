# DevExtreme React Playground

## Run Examples

1. Install npm packages:

    ```bash
    npm install
    ```
2. Generate components and build scripts:

    ```bash
    npm run build
    npm run build:vue
    ```

    or

    ```bash
    npm run build:vue:watch
    ```

## Compile and Hot Reload an Application

```bash
cd ./playground/vue
npm install
npm run serve
```
