1) Launch Google Chrome with the remote debugging port - 9223 
--remote-debugging-port=9223
(https://www.chromium.org/developers/how-tos/run-chromium-with-flags)
 
Open localhost:9223 to check the result. This shows the list of opened browser tabs.
 
Note:
We recommend to use Canary for running these tests.
 
2) Enable Cross Domain Request
Note:
You can use the Cors-Toggle extension for this.
https://chrome.google.com/webstore/detail/cors-toggle/jioikioepegflmdnbocfhgmpmopmjkim
 
3) Run tests
